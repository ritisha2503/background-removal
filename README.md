# Background Removal Tool 🪄🖼️

A simple Python-based background removal project that removes the background from live webcam and generates clean transparent outputs. Built using computer vision and image processing libraries, this project makes it easy to isolate foreground objects with just a single command.

---

## Features ✨

* Remove backgrounds from images automatically
* Generate transparent PNG outputs
* Simple and lightweight implementation
* Fast processing
* Beginner-friendly code structure
* Works on portraits, objects, and general images

---

## Tech Stack ⚙️

* Python
* OpenCV
* NumPy
* PIL (Pillow)
* rembg / image processing libraries

---

---

## Installation 🚀

Clone the repository:

```bash
git clone https://github.com/ritisha2503/background-removal.git
cd background-removal
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## Usage ▶️

Place your image inside the `input/` folder and run:

```bash
python task2.py
```

The processed image will be saved in the `output/` folder with the background removed.
