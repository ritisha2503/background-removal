# Real-Time Background Removal 🎥🪄

A real-time background removal application built using Python and computer vision libraries. This project captures live webcam footage, detects the foreground subject, and removes the background instantly to create a clean segmented video feed.

Perfect for experimenting with computer vision, virtual backgrounds, and live image processing pipelines.

---

## Features ✨

* Real-time webcam background removal
* Live foreground segmentation
* Smooth video frame processing
* Lightweight and fast implementation
* Beginner-friendly project structure
* Supports virtual background experimentation

---

## Tech Stack ⚙️

* Python
* OpenCV
* NumPy
* MediaPipe / rembg / segmentation libraries
* Webcam video processing

---

## How It Works 🧠

```text
Webcam Feed → Frame Processing → Foreground Segmentation → Background Removed Output
```

The application continuously captures frames from the webcam, performs foreground segmentation, and removes or masks the background in real time.

---

## Installation 🚀

Clone the repository:

```bash
git clone https://github.com/ritisha2503/background-removal.git
cd background-removal
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## Usage ▶️

Run the Python script:

```bash
python main.py
```

Your webcam will open and display the live background-removed output.

Press:

```text
q
```

to quit the application.
